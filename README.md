### CSGO bot大战参考代码

#### 如何使用

1.调试好游戏环境
    游戏窗口化
    游戏切换为de_cache地图的死亡竞赛经典模式
    控制台输入 mp_roundtime 60; mp_restartround 1;
2.开启bot
    在pycharm中直接运行main.py
    或python main.py



#### 依赖
    建议py3.7版本
    pip install numpy
    pip install pymem
    pip install pywin32
    pip install opencv-python
